## 📋 תיאור

**Location Tracker** הוא אתר מודרני המאפשר:

✅ **מעקב מיקום בזמן אמת** - עקוב אחרי משתמשים אחרים על מפה  
✅ **תקשורת בזמן אמת** - צ'ט מיידי בין משתמשים דרך WebSocket  
✅ **תמלול קול** - שלח הודעות קול עם תמלול אוטומטי  
✅ **חישוב מרחקים** - חשב מרחק בין שתי נקודות כלשהן  
✅ **מפות OpenStreetMap** - מפות חינמיות ופתוחות  
✅ **ממשק קל** - עיצוב מדוד וקל להשתמש  

## 🛠 טכנולוגיות

### Frontend
- **HTML5** - מבנה האתר
- **CSS3** - עיצוב מדוד ותגובתי
- **Vanilla JavaScript** - ללא תלויות חיצוניות גדולות
- **Leaflet** - ספריית מפות מודרנית
- **OpenStreetMap** - שרת מפות חינמי
- **Web Geolocation API** - קבלת מיקום GPS
- **Web Speech API** - תמלול קול

### Backend
- **Node.js** - סביבת ריצה JavaScript
- **Express** - פריימוורק ווב מהיר
- **WebSocket (ws)** - תקשורת בזמן אמת דו-כיוונית
- **UUID** - יצירת מזהים ייחודיים

## 🚀 התחלה מהירה

### דרישות מוקדמות
- Node.js 18+ ([הורד כאן](https://nodejs.org/))
- NPM או Yarn
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)

### שלבים התקנה

1. **התקן תלויות:**
```bash
npm install
```

2. **הפעל את השרת:**
```bash
npm start
```

השרת יפעל ב-`http://localhost:3000`

3. **פתח בדפדפן:**
- פתח `http://localhost:3000` בדפדפן שלך
- המתן לאישור הרשאות המיקום (GPS)

## 💡 איך להשתמש

### 1️⃣ מעקב מיקום
- האתר יבקש אישור למיקום - לחץ **אישור**
- המיקום שלך יופיע על המפה כנקודה כחולה
- משתמשים אחרים יופיעו אוטומטית על המפה

### 2️⃣ צ'ט בזמן אמת
- הקלד הודעה בתיבת הטקסט
- לחץ **שלח** או הקש Enter
- ההודעה תגיע לכל המשתמשים בזמן אמת

### 3️⃣ הקלטת קול
- לחץ על כפתור **🎤 הקלט**
- דבר בקול ברור
- לחץ **⏹️ עצור** כדי לסיים
- ההודעה תתומלל ותישלח לכל המשתמשים

### 4️⃣ חישוב מרחקים
1. עבור לטאב **📍 מיקום**
2. הכנס רוחב (Latitude) ואורך (Longitude) של היעד
3. לחץ **חשב מרחק**
4. ראה את המרחק בקילומטרים

### 5️⃣ הגדרות
1. עבור לטאב **⚙️ הגדרות**
2. הכנס שם משתמש חדש
3. לחץ **עדכן שם**

## 📡 API WebSocket

### סוגי הודעות

**🔹 שליחה למשתמש אחד:**

```javascript
// עדכון מיקום
{
  "type": "location",
  "lat": 32.0853,
  "lng": 34.7818,
  "accuracy": 10.5
}

// הודעת צ'ט
{
  "type": "chat",
  "text": "שלום לכולם!"
}

// בקשת חישוב מרחק
{
  "type": "distance_request",
  "from": { "lat": 32.0853, "lng": 34.7818 },
  "to": { "lat": 32.1, "lng": 34.8 }
}
```

**🔹 קבלה מהשרת:**

```javascript
// רשימת משתמשים פעילים
{
  "type": "client_list",
  "clients": [
    { "id": "uuid-123", "username": "משתמש 1", "hasLocation": true },
    { "id": "uuid-456", "username": "משתמש 2", "hasLocation": false }
  ]
}

// עדכון מיקום משתמש אחר
{
  "type": "location_update",
  "clientId": "uuid-123",
  "username": "משתמש 1",
  "location": { "lat": 32.0853, "lng": 34.7818, "accuracy": 10.5, "timestamp": "..." }
}

// תוצאת מרחק
{
  "type": "distance_result",
  "distance": "12.50",
  "unit": "km"
}
```

## 🐛 פתרון בעיות

### "לא ניתן לקבל את המיקום שלך"
- ודא שהרשאות GPS מופעלות בדפדפן
- בדוק אם הדפדפן שלך כולל גישה למיקום
- נסה להטעין מחדש את העמוד

### "אין חיבור לשרת"
- ודא שהשרת פועל (`npm start`)
- בדוק שמחברת על `http://localhost:3000`
- בדוק את ה-console (F12) לשגיאות

### "תמלול הקול לא עובד"
- בדוק אם הדפדפן תומך בـ Web Speech API
- תן הרשאות מיקרופון לדפדפן
- ודא שמדברת בעברית בבירור
